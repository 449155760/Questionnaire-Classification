## 快速开始

### 第一步：准备测试数据
1. 将您的测试数据保存为Excel文件，命名为`test.xlsx`
2. 数据格式必须与训练数据`data.xlsx`完全一致
3. 测试数据应包含以下特征（顺序不限）：
   - 人口学特征：性别、年龄、BMI、文化程度等
   - 临床特征：主诊断、手术方式、慢病共存等
   - 量表评分：社会支持评定量表、出院指导质量
   - 出院准备度维度：个人状态(1-3)、适应能力(4-8)、预期性支持(9-12)

### 第二步：运行预测工具
1. 双击运行`predict_gui.exe`
2. 点击"选择测试文件"按钮，选择您的`test.xlsx`文件
3. 点击"开始预测"按钮
4. 等待预测完成，结果将自动保存到同一目录下

## 预测结果
预测结果将保存为新的Excel文件，文件名格式为：`预测结果_YYYYMMDD_HHMMSS.xlsx`

结果文件中将包含：
- 所有原始数据
- 新增"预测类型"列（1, 2, 3）
- 新增"预测类型描述"列（类型描述）

## 类型说明
- **类型1**: 低准备度 - 需要更多的出院准备支持
- **类型2**: 中准备度 - 中等准备度，需要部分支持
- **类型3**: 高准备度 - 准备充分，可顺利出院

## 注意事项
1. 确保测试数据格式与训练数据一致
2. 如有缺失值，系统会自动填充
3. 预测结果仅供参考，实际决策应结合临床判断
4. 如需更新模型，请联系系统管理员

## 技术支持
如有问题，请检查：
1. 测试文件是否为Excel格式（.xlsx）
2. 是否包含必要的特征列
3. 数据中是否有异常值
